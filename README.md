# Humanitarian Distribution Dashboard

An interactive Power BI dashboard developed to analyze humanitarian beneficiary distribution data.

## Features

- Total Households KPI
- Total Individuals KPI
- Distribution Rate
- Household Size Analysis
- Category Analysis
- Distribution Status
- Distribution Timeline
- Interactive Filters

## Tools Used

- Microsoft Power BI
- DAX
- Microsoft Excel

## Purpose

This dashboard demonstrates data analysis and visualization skills for humanitarian operations, supporting evidence-based decision-making through interactive reports.
